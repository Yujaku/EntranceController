# EntranceController
Unityをクライアントとし、Slackに入退室状況を反映する入退室管理アプリ  
developブランチにpushしてください

## 現在できていること
* Slackへのメッセージ送信機能
* 入力したテキストデータのデバック表示.(string型メソッドで取得可能にしています)  
* Toggleを使った班ごとの入退室状況の反映.  


## 未実装の範囲(必須)
